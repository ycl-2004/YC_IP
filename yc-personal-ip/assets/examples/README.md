# 参考图分类（生图时怎么用）

参考图只锁两件事：**①YC 长相 ②画风渲染**。锁完不照抄构图——每次从当前内容重想画面。

## sticker/（默认模式用）
- **`00-reference-sheet-watercolor.png` = canonical 画风锁。每张 Sticker 固定带这一张，别换。** 它锁住"白底水彩铅笔松手绘"这个画风，是 Sticker 不跑成海报的命门。
- `01-08-*.png`：动作姿势库（坐着用笔记本、托腮思考、拿杯子…）。**只作构思参考**——想画某个动作先翻它找最接近的，不当画风锁。
- `outfit-01~09.png`：服装变体参考。**只作构思参考**，不当画风锁。

> Sticker 生图 = 永远带 00；红发/眼镜不稳时再加 `character-reference/` 九宫格一起带。

## simple/（仅多步流程用）
01-07 是 Simple 模式例图（流程流、桌面场景、before/after 等）。用对应内容最接近的一张作画风+密度校准。

## character-reference/（长相锁，跨模式）
01-07 九宫格 = YC 长相锁。注意它是**全彩 anime，只能锁长相、锁不了水彩画风**——所以 Sticker 要 `00 + 九宫格`两张一起带。`05/06/07-pose-library-*` 是几十个动作的姿势库。

## rich/（已不在默认路径）
仅"整页 IP 介绍/品牌全景设定页"这类极少数场景手动启用时才参考。日常忽略。
