# YC 视觉风格 DNA

## 一句话（按模式分）

YC 的视觉风格是一个**从轻到重的光谱**，不是单一风格：

- **Sticker 贴图（最轻，最常用）**：纯白底上一张水彩铅笔手绘的小贴纸。像把 YC 从笔记本边角剪下来贴在白纸上。轻、干净、留白多、几乎没有文字。**这是默认形态**。
- **Simple 普通**：温暖的视觉故事流，3-6 步讲清一件事。
- **Rich 丰富**：温暖手账 editorial，把想法铺在一张漂亮的笔记页上（信息最满）。

> ⚠️ **防跑偏第一定律**：模型的默认先验是"精致全彩 anime 插画 + 满版拼贴"，也就是**海报**。如果不主动压制（白底、水彩铅笔松线、删道具、删装饰、带参考图），任何模式都会自动滑向海报。Rich 是唯一允许"满"的模式，Sticker/Simple 必须主动留白。

不是商业海报，不是 PPT，是有温度的创作者内容。

## 背景与基调

- 背景：奶油白 / 米白（warm off-white），不是纯白，不是灰色
- 轻微纸感质感可以有，但不要过重
- 整体色调偏暖：红、牛仔蓝、软粉、奶油色、灰
- 手绘与排版混合：section header 用小标题排版，内容标注用手写风格

## 必须有的元素

- **YC 角色**：chibi 比例，红发，透明/细框眼镜（必须出现）
- **MacBook**：始终带一个小心形贴纸 ♥ （signature prop）
- **笑脸咖啡杯**：smiley face mug ☺，工作/创意场景的标配
- **签名装饰**：小心形 ♥、音符 ♪、星星 ✦、闪光 ✨（克制使用，点缀用）
- **中英混排**：标注同时有中文和英文，格式如 `music lover 热爱音乐`
- **section header**：用细线下划线 + 全大写英文标题（如 `ABOUT YC`、`LIFE MODE ♥`）

## 色板

| 颜色 | 用途 |
|------|------|
| **深红/酒红** | YC 头发、品牌 logo 字、强调色 |
| **牛仔蓝** | 系统/流程箭头、section 下划线、技术类标注 |
| **黑色** | 主体线稿、主要文字 |
| **奶油白** | 背景底色 |
| **软粉色** | 心形装饰、浪漫/情感类标注、便条纸底色 |
| **灰色** | 次要文字、便条纸边框、辅助线 |

蓝色用于技术/系统内容；粉色用于情感/生活内容；红色是 YC 的 signature 颜色。

## 签名道具清单

根据场景选 2-4 个：

| 道具 | 场景 |
|------|------|
| MacBook（♥ 贴纸） | 工作、编程、写作、内容创作 |
| 笑脸咖啡杯 ☺ | 所有工作/创意场景 |
| MIDI 键盘 | 音乐创作场景 |
| 头戴耳机 | 音乐/专注场景 |
| 螺旋笔记本 / 硬封面笔记本 | 想法/规划/学习场景 |
| 铅笔杯（装满笔） | 创作场景 |
| 小植物/绿植 | 背景装饰，生活/咖啡馆场景 |
| 书籍（硬封面叠放） | 学习/成长场景 |
| 背包 | 出行/留学/户外场景 |

## 布局规则

- 多面板布局：大图有多个 section card，每个 card 有标题 + 内容
- Section card 用细圆角边框或便条纸样式
- 内容层级：大标题（全大写）→ 小标题（首字大写）→ 正文标注
- YC 角色通常居中或左侧，占画面 30%-50%
- 留白要有节奏：不是随机留白，而是每个 section 有呼吸感

## 绝对不要

- 纯白背景（要奶油白/米白）
- 过度商业插画感（精修、过亮、饱和度太高）
- 正式流程图 / PPT 信息图
- 纯英文或纯中文（必须中英混排）
- YC 缺少红发或眼镜
- 过度可爱/幼稚（保持温暖但成熟）
- 没有签名道具（特别是 MacBook 和笑脸杯）

## 三模式风格对比

### Sticker 贴图模式（原 Minimal，默认）

目标是**一张小贴纸**，不是一张图说，更不是海报。参考基准：白底上单个 YC 角色的水彩铅笔手绘小图。

- **背景**：纯白 #FFFFFF（不是奶油白、不是纸纹、不是渐变、不是阴影）
- **画风**：**水彩铅笔 / 彩色铅笔轻涂**手绘感 —— 线条松、有轻微抖动、上色淡、留白透气。**不是**精致全彩 anime，**也不是**纯黑线稿
- **YC**：**保留红发可识别**，但是松手绘的轻渲染版（soft watercolor-pencil），不是高光厚涂的成品插画
- **主体**：**一个** YC + 最多一个简单物件/动作。单主体，不拼贴、不分格、不串联多场景
- **颜色**：以 YC 红发为主色锚点，配淡淡的暖色(米黄/软蓝/软粉点缀)，整体低饱和、轻
- **文字**：0-3 个中文短词（能不写就不写）；无英文、无中英混排、无 section header
- **装饰**：零或极少。无满天心形/星星/闪光、无便利贴墙、无大"YC"logo、无书堆笔筒植物围一圈
- **道具**：最多 1 个，且必须和这张图的意思直接相关（不是为了"signature"硬塞 MacBook/笑脸杯）
- **留白**：至少 50% 画面是空白
- **审美**：像从笔记本边角剪下来贴在白纸上的一张小贴纸 —— 轻、干净、有手感
- **必须带参考图**：生图时附上贴纸参考图（见 prompt-template），纯文字 prompt 会跑成海报

### Simple 普通模式

- **背景**：奶油白，轻微纸感
- **画风**：温暖 chibi 插画，多步骤场景
- **YC**：每一步出现，不同动作/状态
- **颜色**：黑 + 暖色（橙/红/粉/蓝适量）
- **文字**：每步 1-2 个中英混排标注
- **装饰**：少量心形/星星
- **道具**：2-3 个签名道具
- **审美**：像一个温暖的视觉故事流

### Rich 丰富模式

- **背景**：奶油白，纸感
- **画风**：多面板 editorial，完整信息架构
- **YC**：中心或主要位置，精致版
- **颜色**：完整色板（红/蓝/粉/灰都用）
- **文字**：中英混排，section header，多标注
- **装饰**：心形、星星、闪光、便条纸
- **道具**：3-4 个签名道具
- **审美**：像认真年轻人的创作笔记页
