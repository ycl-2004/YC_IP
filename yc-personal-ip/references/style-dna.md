# YC 视觉风格 DNA

## 一句话

温暖手账 editorial。像一个认真生活、浪漫创作的年轻人把自己的想法铺在一张漂亮的笔记页上。

不是极简草图，不是商业海报，不是 PPT，是有温度的创作者内容。

## 背景与基调

- 背景：奶油白 / 米白（warm off-white），不是纯白，不是灰色
- 轻微纸感质感可以有，但不要过重
- 整体色调偏暖：红、牛仔蓝、软粉、奶油色、灰
- 手绘与排版混合：section header 用小标题排版，内容标注用手写风格

## 必须有的元素

- **YC 角色**：chibi 比例，红发，透明/细框眼镜（必须出现）
- **MacBook**：始终带一个小心形贴纸 ♥ （signature prop）
- **笑脸咖啡杯**：smiley face mug ☺，工作/创意场景的标配
- **签名装饰**：小心形 ♥、音符 ♪、星星 ✦、闪光 ✨（克制使用，点缀用）
- **中英混排**：标注同时有中文和英文，格式如 `music lover 热爱音乐`
- **section header**：用细线下划线 + 全大写英文标题（如 `ABOUT YC`、`LIFE MODE ♥`）

## 色板

| 颜色 | 用途 |
|------|------|
| **深红/酒红** | YC 头发、品牌 logo 字、强调色 |
| **牛仔蓝** | 系统/流程箭头、section 下划线、技术类标注 |
| **黑色** | 主体线稿、主要文字 |
| **奶油白** | 背景底色 |
| **软粉色** | 心形装饰、浪漫/情感类标注、便条纸底色 |
| **灰色** | 次要文字、便条纸边框、辅助线 |

蓝色用于技术/系统内容；粉色用于情感/生活内容；红色是 YC 的 signature 颜色。

## 签名道具清单

根据场景选 2-4 个：

| 道具 | 场景 |
|------|------|
| MacBook（♥ 贴纸） | 工作、编程、写作、内容创作 |
| 笑脸咖啡杯 ☺ | 所有工作/创意场景 |
| MIDI 键盘 | 音乐创作场景 |
| 头戴耳机 | 音乐/专注场景 |
| 螺旋笔记本 / 硬封面笔记本 | 想法/规划/学习场景 |
| 铅笔杯（装满笔） | 创作场景 |
| 小植物/绿植 | 背景装饰，生活/咖啡馆场景 |
| 书籍（硬封面叠放） | 学习/成长场景 |
| 背包 | 出行/留学/户外场景 |

## 布局规则

- 多面板布局：大图有多个 section card，每个 card 有标题 + 内容
- Section card 用细圆角边框或便条纸样式
- 内容层级：大标题（全大写）→ 小标题（首字大写）→ 正文标注
- YC 角色通常居中或左侧，占画面 30%-50%
- 留白要有节奏：不是随机留白，而是每个 section 有呼吸感

## 绝对不要

- 纯白背景（要奶油白/米白）
- 过度商业插画感（精修、过亮、饱和度太高）
- 正式流程图 / PPT 信息图
- 纯英文或纯中文（必须中英混排）
- YC 缺少红发或眼镜
- 过度可爱/幼稚（保持温暖但成熟）
- 没有签名道具（特别是 MacBook 和笑脸杯）

## 三模式风格对比

### Minimal 极简模式

- **背景**：纯白（不是奶油白！纯白 #FFFFFF）
- **画风**：黑色线稿草图感，不是精致 anime 插画
- **YC**：简化线稿版，不是全彩精修版
- **颜色**：黑 + 唯一一种强调色（橙或红）
- **文字**：最多 2-5 个中文词，无英文
- **装饰**：零。无心形、无星星、无闪光、无植物
- **道具**：无签名道具，除非道具本身就是隐喻
- **审美**：像在白纸上用笔快速画了一个想法

### Simple 普通模式

- **背景**：奶油白，轻微纸感
- **画风**：温暖 chibi 插画，多步骤场景
- **YC**：每一步出现，不同动作/状态
- **颜色**：黑 + 暖色（橙/红/粉/蓝适量）
- **文字**：每步 1-2 个中英混排标注
- **装饰**：少量心形/星星
- **道具**：2-3 个签名道具
- **审美**：像一个温暖的视觉故事流

### Rich 丰富模式

- **背景**：奶油白，纸感
- **画风**：多面板 editorial，完整信息架构
- **YC**：中心或主要位置，精致版
- **颜色**：完整色板（红/蓝/粉/灰都用）
- **文字**：中英混排，section header，多标注
- **装饰**：心形、星星、闪光、便条纸
- **道具**：3-4 个签名道具
- **审美**：像认真年轻人的创作笔记页
