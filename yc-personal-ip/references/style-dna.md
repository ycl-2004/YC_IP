# YC 视觉风格 DNA

## 一句话

YC 的视觉是**两档密度**，默认偏轻：

- **Sticker 贴图（默认，最常用）**：纯白底上一张水彩铅笔手绘小贴纸。像把 YC 从笔记本边角剪下来贴在白纸上——轻、干净、留白多、几乎没文字。
- **Simple 普通（仅多步流程）**：奶油白底上的温暖视觉故事流，3-6 步讲清一件事。

> ⚠️ **防跑偏第一定律**：模型默认先验是"全彩 anime + 满版拼贴"=海报。不主动压制（白底/奶油白、水彩铅笔松线、删多余道具、带参考图），两档都会滑向海报。**怀疑时永远选更轻的 Sticker。**

不是商业海报，不是 PPT，是有温度的创作者内容。

---

## Sticker 贴图模式（默认）

目标是**一张小贴纸**，不是插画、更不是海报。基准：白底上单个 YC 的水彩铅笔手绘小图（`assets/examples/sticker/00-reference-sheet-watercolor.png`）。

- **背景**：纯白 #FFFFFF（不是奶油白、不是纸纹、不是渐变阴影）
- **画风**：水彩铅笔/彩色铅笔轻涂——线松、有轻微抖动、上色淡、透气。不是精致全彩 anime，也不是纯黑线稿
- **YC**：保留红发可识别，但松手绘轻渲染，不是高光厚涂成品
- **主体**：一个 YC + 最多一个简单物件/动作。不拼贴、不分格、不串联
- **颜色**：红发为主色锚点，配淡暖色点缀（米黄/软蓝/软粉），整体低饱和
- **文字**：0-3 个中文短词（能不写就不写）；无英文、无中英混排、无 section header
- **装饰**：零或极少。无满天心形/星星、无便利贴墙、无大"YC"logo、无道具围一圈
- **道具**：最多 1 个且和这张图的意思直接相关——**不为"signature"硬塞 MacBook/笑脸杯**
- **留白**：至少 50%
- **必带参考图**：00 画风锁（长相不稳时加 character-reference 九宫格）

---

## Simple 普通模式（仅多步流程）

只有"非得分 3-6 步才说得清"时才用。这一档才适用下面这些温暖手账元素：

- **背景**：奶油白 / 米白（warm off-white），轻微纸感，不是纯白
- **画风**：温暖 chibi 插画，多步骤场景，每步 YC 不同动作
- **签名道具**：选 2-3 个（见下表）。工作/创意场景常带 **MacBook（♥贴纸）** 和 **笑脸咖啡杯 ☺**
- **文字**：每步 1-2 个中英混排标注，格式如 `music lover 热爱音乐`
- **装饰**：少量心形 ♥ / 星星 ✦（克制）
- **连接**：箭头或流动路径串联，**不要**重型面板边框和 section header（那会变满版 editorial）

### 色板
| 颜色 | 用途 |
|------|------|
| 深红/酒红 | YC 头发、品牌强调色 |
| 牛仔蓝 | 流程箭头、技术类标注 |
| 黑色 | 线稿、主要文字 |
| 奶油白 | Simple 背景底色 |
| 软粉色 | 心形装饰、情感类标注 |
| 灰色 | 次要文字、辅助线 |

蓝=技术/系统，粉=情感/生活，红=YC signature。

### 签名道具清单（Simple 用，按场景选 2-3）
| 道具 | 场景 |
|------|------|
| MacBook（♥贴纸） | 工作、编程、写作、创作 |
| 笑脸咖啡杯 ☺ | 工作/创意场景 |
| MIDI 键盘 / 头戴耳机 | 音乐/专注 |
| 螺旋/硬封面笔记本 | 想法、规划、学习 |
| 铅笔杯 | 创作 |
| 小植物 | 生活/咖啡馆 |
| 书籍叠放 | 学习/成长 |
| 背包 | 出行/留学/户外 |

---

## 绝对不要（两模式通用）
- YC 缺红发或眼镜 / 纯黑框眼镜（应是透明细框）
- 过度商业插画感（精修、过亮、饱和过高）
- 正式流程图 / PPT 信息图
- 过度可爱幼稚（温暖但成熟）
- **把 Simple 的元素塞进 Sticker**（奶油白底、MacBook/笑脸杯、中英混排、section header）——这是 Sticker 滑向海报的最常见原因
