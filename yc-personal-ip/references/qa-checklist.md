# QA Checklist — YC

## 通用必过项（两模式都要）
- [ ] YC 有红发（深红/酒红色）
- [ ] YC 有透明/细框圆眼镜（不是黑框）
- [ ] YC 参与了核心动作，不只是装饰
- [ ] 比例符合用户需求
- [ ] 带了参考图，没有照抄旧图的具体布局

角色识别失败（缺红发/眼镜）= 最高优先级，必须重生成：强调 `messy dark-red burgundy hair AND round clear-frame glasses — both mandatory`。

---

## Sticker 贴图模式专项（默认）

- [ ] 背景是**纯白 #FFFFFF**（不是奶油白、不是纸纹、不是渐变阴影）
- [ ] 画风是**水彩铅笔轻涂松手绘**（不是高光全彩 anime，也不是纯黑线稿）
- [ ] **只有一个 YC**，单主体，无多场景/分格/步骤流
- [ ] 全图 0-3 个中文短词，无英文、无中英混排、无 section header
- [ ] 至少 50% 留白
- [ ] 最多 1 个相关物件；无便利贴墙、无大"YC"logo、无书堆/笔筒/植物围一圈
- [ ] 无满天心形/星星/闪光
- [ ] 像从笔记本剪下来贴在白纸上的小贴纸

**Sticker 失败信号（都是滑向海报）：**
| 信号 | 解决 |
|------|------|
| 背景奶油白/纸纹 | 重生成，强调 PURE WHITE #FFFFFF |
| 画得太精致全彩高光 | 强调 loose watercolor colored-pencil sketch, not glossy anime |
| 一圈道具/便利贴/书堆/大 logo | 删到只剩单角色 + 最多 1 物件 |
| 出现英文或中英混排 | 重生成，只要 0-3 个中文短词 |
| 出现多场景/箭头流程 | 那是 Simple，压回单主体 |
| 没带参考图 | 补 `00-reference-sheet-watercolor` 重生成（最常见根因）|

---

## Simple 普通模式专项（仅多步流程）

- [ ] 背景是奶油白/米白（不是纯白）
- [ ] 3-6 个步骤/场景串联，每步 YC 不同动作
- [ ] 每步标注短（2-6 字），中英混排
- [ ] 步骤间有箭头或流动路径连接
- [ ] 没有重型面板边框 / section header（那会变成满版 editorial）
- [ ] 整体像一个温暖的视觉故事

**Simple 失败信号：**
| 信号 | 解决 |
|------|------|
| 背景是纯白 | 强调 warm off-white cream background |
| 变成多面板 + section header | 去掉框和标题，回到 vignette 串联 |
| 步骤太多（>6）| 精简到核心 3-4 步 |
| 文字太密 | 每步只保留 1 个最核心标注 |
| 太像 PPT/流程图 | 强调 hand-drawn warm story, not infographic |

---

## 迭代优先级
1. 角色识别（红发/眼镜）→ 必重生成
2. 背景颜色（Sticker 要纯白 / Simple 要奶油白）→ 重生成
3. 信息密度（Sticker 太满 / Simple 太密）→ 删元素重生成
4. 比例不对 → 检查提示词比例参数
5. 文字问题（错字/英文混入 Sticker）→ 局部编辑或重生成

## 交付标准
- **Sticker**：白底一张水彩铅笔小贴纸，单角色、极少字、巨量留白，1 秒看懂。
- **Simple**：温暖视觉故事流，3-6 步讲清一件事。
- 两者都要：YC 有红发+眼镜。
