# 使用示例

## 基础用法

### 配图策略（shot list，不生图）

```text
/yc-personal-ip 先不要生图。
分析下面这篇文章，输出 shot list，建议在哪些段落配图。
每张图写清楚：段落位置、主题、核心意思、布局类型、YC 在做什么、建议比例、建议标注词。

<粘贴文章>
```

### 文章配图（默认 16:9）

```text
/yc-personal-ip 为下面这篇文章生成 4 张 16:9 配图，YC 风格。

<粘贴文章>
```

### 指定比例

```text
/yc-personal-ip 生成一张 9:16 的竖版 Story 图，关于「用 AI 工具把效率翻倍」这个主题。
```

```text
/yc-personal-ip 生成一张 1:1 的小红书方图，YC 坐在咖啡馆工作，感觉专注又有点 chill。
```

```text
/yc-personal-ip 生成一张 21:9 的超宽 banner，展示 YC 的个人品牌生态和5大内容支柱。
```

## IP 介绍类

### 自我介绍主图

```text
/yc-personal-ip 生成一张 16:9 的 YC IP 介绍图。
展示 YC 是谁、关注什么主题、有什么特点。
用 IP Character Sheet 布局，包含 ABOUT YC、FAVORITE THINGS、LIFE MODE 几个部分。
```

### 品牌生态地图

```text
/yc-personal-ip 生成一张 16:9 的 YC 个人品牌生态图。
用 Brand Ecosystem Map 布局，YC 居中，5大内容支柱环绕四周。
```

### Profile Banner

```text
/yc-personal-ip 生成一张 21:9 的 Profile Banner。
YC 在画面中间，左右各有关键词标注，整体简洁大气。
```

## 内容支柱类

### AI / 技术工具

```text
/yc-personal-ip 生成一张 16:9 的图，主题是「用 AI 把混乱的想法变成结构化项目」。
用 Process Transformation 布局，左边是散乱的 sticky note，右边是清晰的项目结构，YC 在中间拿着 Define Goal 的纸条。
```

### 个人成长 / 学习

```text
/yc-personal-ip 生成一张 16:9 的图，讲「每天输入一点，时间久了会发生什么」。
YC 把信息一点点装进某个容器，容器慢慢变满，标注：积累 / 复利 / Keep Learning。
```

### 音乐创作

```text
/yc-personal-ip 生成一张 16:9 的图，展示 YC 的音乐创作系统。
用 Deep Work Journal 布局，YC 在工作室，周围展示创作流程：情绪 → 歌词 → 旋律 → 编曲 → 录音 → 分享。
```

### 留学 / 生活

```text
/yc-personal-ip 生成一张 4:5 的小红书配图，YC 背着包在城市街头，感觉轻松 chill。
标注：City Walks / 探索中 / Good Days。
```

## 知识讲解类（任意主题）

### 讲解技术概念

```text
/yc-personal-ip 用 YC 的风格生成一张 16:9 的图，讲解「什么是递归」。
用 Knowledge Explainer 布局，YC 在黑板前，黑板上有递归的简单示例，旁边有中英混排标注。
```

### 讲解数学概念

```text
/yc-personal-ip 生成一张 16:9 的图，YC 来讲解「复利的力量」。
黑板上有一个简单的增长曲线图，YC 指着曲线加速的那个拐点，标注：时间 / 坚持 / 指数增长。
```

### 讲解思维框架

```text
/yc-personal-ip 生成一张 16:9 的图，YC 讲解「第一性原理思维」。
用便条纸/卡片排列，YC 把复杂问题拆解到最基础的原则，再往上重建。
```

## 编辑和修复

### 去掉多余标题

```text
/yc-personal-ip 帮我编辑这张图，去掉左上角的「流程图」文字，其他内容保持不变。
```

### 角色识别修复

```text
/yc-personal-ip 重新生成这张图。要求角色必须有：蓬松深红发色 + 透明圆框眼镜。其他构图保持一样。
```

### 调整比例

```text
/yc-personal-ip 把上面这张图的概念重新生成一版 9:16 竖版的。
```
