# 构图模式 — YC 版

## YC 的7种专属布局类型

### 类型 1：IP Character Sheet（角色设定页）

**适合**：自我介绍、IP 介绍、角色说明

**结构**：
- 左侧：YC 站立全身图（大图）+ ABOUT YC 标签列表
- 中间：FAVORITE THINGS 道具图示 + 标注
- 右侧：LIFE MODE 4格小场景（每格一个生活状态）
- 下方：OUTFIT VIBES 4套服装小图 + COLOR PALETTE
- 签名便条：signature quote

**推荐比例**：16:9 横版（内容最丰富）或 21:9 超宽版

---

### 类型 2：Creative Workspace Panorama（创意工作台全景）

**适合**：深度内容介绍、工作方式展示、"Build in Public" 类内容

**结构**：
- YC 居中坐在桌子前（MacBook、笑脸杯、笔记本、植物）
- 左侧：2-3 个 concept card（流程/价值观/方法论）
- 右侧：2-3 个 concept card（工具/checklist/结果）
- 上方：一句主题标题
- 下方：底部 tagline

**推荐比例**：16:9 或 21:9（超宽全景感）

---

### 类型 3：Brand Ecosystem Map（品牌生态地图）

**适合**：个人品牌介绍、内容支柱展示、多维度自我介绍

**结构**：
- YC 站在画面中心
- 5个内容支柱以卡片形式环绕（上左/上右/下左/下右/下中）
- 每个支柱有标题 + 3-5 个关键词 + YC 在该场景的小图
- 左侧栏：WHAT YC REPRESENTS + HOW YC CAN BE USED
- 右侧栏：YC CONTENT PILLARS + MISSION
- 底部：signature tagline

**推荐比例**：16:9 或 21:9

---

### 类型 4：Deep Work Journal（深度主题日志）

**适合**：单一内容支柱的深度内容（音乐创作系统、学习方法、项目流程）

**结构**：
- YC 大图占画面 40-50%，正在做该主题的核心动作
- 围绕 YC：相关工具/道具（键盘、书、代码）
- 左侧/右侧：3-4 个内容板块（流程/系统/工具/理念）
- 便条纸/卡片式呈现各模块
- 签名引用语

**推荐比例**：16:9

---

### 类型 5：Process Transformation（过程转化图）

**适合**：Before/After、混乱到有序、想法到产品、成长记录

**结构**：
- YC 居中，拿着/展示核心关键词或纸条
- 左侧：混乱状态（sticky notes 飞散、问号、碎片）
- 右侧：结构化状态（清晰流程、checklist、结果）
- 中间 YC 是转化的关键节点
- 底部小字：identity keywords

**推荐比例**：16:9 或 4:3

---

### 类型 6：Broad Concept Explainer（大概念隐喻解释图）

**适合**：解释“什么是 X / explain X / 一个抽象概念怎么工作”。不限于 YC 的5大内容支柱，任何"YC 来给你解释一下"的场景都适用。

**核心原则**：不要做百科页、课程卡片或白板讲课。只画一个核心隐喻，让观众一眼看到机制。

**结构**：
- 左侧：2-3 组具体、可区分的例子；必要时显示已有答案
- 中间：一个可操作的低科技隐喻装置（学习机、分拣机、压缩机、过滤器、桥、传送带等）
- YC：正在用明确物理动作驱动装置，必须参与核心因果
- 装置内部/出口：出现学到或处理后留下的可见结构（网、筛、轨道、刻度、模板等）
- 右侧：一个新输入经过结构，得到具体判断或结果
- 标注：默认 4-6 个中文短词，必要时加少量箭头

先读取 `concept-director.md` 完成“机制判断 → 认知动作 → 物理动词 → 装置 → YC 职责 → 新输入验证”。四个视觉节拍放在一个连续场景里，不做四格面板。

**知识类型对应道具**：
- 机器学习 → 分好类的水果例子进入手摇学习机，YC 转曲柄把特征织成规则网，再用新水果测试并输出带问号的预测
- 数学概念 → 把公式变成可操作物件，例如天平、盒子、路径、折叠纸
- 编程/技术 → 把代码/输入变成小零件，YC 送入调试机器或转换装置
- 方法论 → 把方法画成可操作的场景物件，YC 正在里面操作

**禁止**：
- 大标题 + 定义段落 + 分类列表 + 真实案例 + 总结条
- 正式白板课件 / PPT / textbook infographic
- YC 只是站在旁边指向图表
- 中英混排说明堆满画面

**推荐比例**：16:9（默认）或 4:3

---

### 类型 7：Social Media Card（社媒单张）

**适合**：单条小红书帖、朋友圈、Instagram 单张

**结构**：
- YC 在一个清晰场景（咖啡馆/书桌/户外）
- 1-2 个主要标注
- 1句 signature quote
- 保持简洁，信息密度低于前5种

**推荐比例**：1:1（方图）或 4:5（小红书竖版）或 9:16（Story）

---

## 比例 × 布局选择建议

| 比例 | 最适合的布局类型 |
|------|------|
| 16:9 | Workspace Panorama, Ecosystem Map, Deep Work Journal, Process Transformation |
| 21:9 | IP Character Sheet, Workspace Panorama（超宽版） |
| 1:1 | Social Media Card（通用） |
| 4:5 | Social Media Card（小红书竖版） |
| 9:16 | Social Media Card（Story / 竖版海报） |
| 4:3 | Process Transformation, Deep Work Journal |
| 3:2 | 文章配图 |
| 2:3 | 竖版海报、Pinterest |
| 5:4 | 微博图 |

---

## 原创规则

每次从当前内容发明新的场景组合，不要照搬已有图的具体布局。

- 道具可以重复（MacBook、笑脸杯是签名必须出现）
- 但场景的具体构图、卡片内容、YC 的姿势都要根据当前主题重新设计
- Signature quotes 选与当前主题最匹配的，不要每张都用同一句
