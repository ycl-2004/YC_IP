# YC IP 角色

## 角色定义

YC 是 YC 个人 IP 内容配图的固定视觉角色。每张图默认出现 YC。YC 必须参与图里的核心场景或动作，不能只是站着看。

## 外形（必须一致）

### 核心识别特征（缺一不可）

- **头发**：蓬松凌乱的深红/酒红色短发 — 最重要的识别标志
- **眼镜**：圆形透明/细框眼镜（clear frame glasses）
- **眼睛**：大而有神，anime chibi 比例
- **脸型**：小而柔和，圆润
- **比例**：chibi — 头大身小，细手细腿

### 配件（经常出现）

- 小耳环（单侧）
- 银色链条项链
- 银色戒指（偶尔）
- 白色运动鞋

### 服装库（按场景选择）

| 套装 | 名称 | 适用场景 |
|------|------|------|
| 牛仔夹克 + 白色 T + 牛仔裤 | Denim Casual 牛仔休闲 | 日常、户外、通用 |
| 黑色外套 + 黑色工装裤 | Black Workwear 工装酷感 | 专注工作、深夜创作、Engineering |
| 棒球外套（Varsity） | Varsity Playful 学院活力 | 校园、轻松、留学场景 |
| 奶油白外套 + 黑色内搭 | Clean Cream 简约清柔 | 创意、咖啡馆、内容创作 |

**默认服装**：Denim Casual（最通用）。音乐场景用 Clean Cream；深夜 coding 用 Black Workwear；户外/留学用 Varsity。

## 生图关键词（必须包含）

### Sample / Standard 模式（默认）

```
chibi character, messy dark-red burgundy hair, round clear-frame glasses, big expressive anime eyes, small gentle round face, chibi proportions large head small body, casual streetwear, small earring, silver chain necklace, white sneakers
```

### Minimal / Sticker 贴图模式（水彩铅笔轻渲染）

Minimal / Sticker 模式要的是**松手绘小贴纸**，不是默认 Sample，也不是精致全彩 anime。保留红发+眼镜识别，但渲染要轻：

```
chibi character YC, messy dark-red burgundy hair (red hair clearly recognizable), round clear-frame glasses, chibi proportions, drawn as a LOOSE watercolor colored-pencil sketch — soft sketchy linework with slight wobble, light low-saturation coloring, hand-drawn and airy, NOT glossy polished anime, NOT heavy shading, single subject on pure white background
```

- 去掉 big expressive anime eyes 的高光精修感，换成简单柔和的眼睛
- 配件（耳环/项链/运动鞋）能省则省，贴图重在轻和识别度，不堆细节
- 红发是唯一需要"明显"的颜色，其余轻轻带过

## 表情/状态库

| 状态 | 描述 | 适用场景 |
|------|------|------|
| 专注 | 微微皱眉，看屏幕 | 工作、编程、写作 |
| 好奇 | 圆眼睛微睁，有问号感 | 探索、学习新东西 |
| 平静思考 | 托腮，望着前方 | 感悟、回顾、规划 |
| 轻松 chill | 嘴角上扬 | 户外、生活、音乐 |
| 创作沉浸 | 专注对着键盘/笔记本 | 音乐制作、写作 |
| 展示/分享 | 拿着或指向某样东西 | 教程类、展示类内容 |

## 常见场景动作

- 对着 MacBook 思考或打字（最常用）
- 戴头戴耳机在电子琴前创作
- 拿着书或在书架前
- 背着包在街上/车站
- 在咖啡馆角落工作（笑脸杯在旁边）
- 拿着/展示某张纸或卡片
- 坐在户外（公园、草地、城市角落）
- 把想法/内容整理进某个系统或容器里

## 禁止

- 缺少红发 → 识别失败
- 缺少眼镜 → 识别失败
- 纯黑框眼镜（应是透明/细框）
- 只站在旁边看，不参与场景
- 过度幼稚卡通（不是儿童角色）
- 表情过度夸张（保持温暖真实感）
- 正式服装（西装、制服）

## 判断标准

去掉 YC，图的核心场景还完全成立 → YC 太装饰了，重写让 YC 参与核心动作。
